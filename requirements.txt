# Pikaia — dependency manifest
# ============================================================
# CORE: zero required third-party packages.
# All LLM providers use Python's stdlib urllib.request — no SDK
# needs to be installed to run Pikaia with any provider.
#
# Install optional groups based on the features you use:
#
#   pip install -r requirements.txt          # core only (no extras)
#   pip install -r requirements-lancedb.txt  # fast vector MT backend
#   pip install -r requirements-deepseek.txt # local DeepSeek-R1 1.5B
#   pip install -r requirements-dev.txt      # diagrams + tooling
#
# Or use pyproject.toml optional groups:
#
#   pip install ".[lancedb]"
#   pip install ".[deepseek]"
#   pip install ".[all]"
# ============================================================
